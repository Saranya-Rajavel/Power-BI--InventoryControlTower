## Overview
This project presents an executive-level Power BI dashboard designed to monitor inventory health, identify stock-out and overstock risks, analyze slow-moving products, and improve inventory turnover across multiple warehouses.

## Business Problems Addressed
- Stock-outs causing delayed customer deliveries
- Overstocking leading to high holding costs
- Lack of data-driven inventory decision-making

## Key Features
- Inventory health monitoring (Low / Optimal / Excess)
- Stock-out & replenishment analysis
- Overstock & slow-moving product identification
- Inventory turnover & performance analysis
- Executive control tower view with navigation

## Tools & Technologies
- Microsoft Power BI
- DAX
- Supply Chain Analytics
